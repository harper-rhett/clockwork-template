# Clockwork Engine Template
This is a template for the game framework [Clockwork](https://github.com/harper-rhett/clockwork-engine). It has a basic example to help you get started on a new project. Install the zip file from [GitHub Releases](https://github.com/harper-rhett/clockwork-template/releases) and then drop the file into `C:\Users\%USERNAME%\Documents\Visual Studio 2022\Templates\ProjectTemplates`.
